# Plan: Data Analytics Consultant Portfolio Website

## Objective
Create a minimalist, high-performance portfolio website for Lookkaew that highlights expertise in technical data execution, strategic consulting, and educational leadership. The site will be optimized for GitHub Pages.

## Visual Strategy: "Minimalist & Clean"
- **Color Palette:** Whitespace-heavy with soft greys and a single accent color (e.g., Deep Navy or Emerald Green) for focus.
- **Typography:** Modern sans-serif (Inter or System UI) for maximum readability.
- **Layout:** Single-page scroll with distinct sections separated by clean borders or subtle shifts in background color.

## Site Structure
1. **Hero Section:** Clear headline: "Lookkaew: Data Analytics Consultant." Sub-headline: "Turning complex data into actionable business strategy and empowered teams."
2. **Services (The Trio):**
   - **Strategy:** Data auditing, roadmap planning, and PDPA compliance.
   - **Execution:** BI Dashboarding (Power BI/Tableau), Data Modeling (SQL/BigQuery), and Python automation.
   - **Education:** Custom workshops and data-driven team upskilling (leveraging Skooldio experience).
3. **Project Portfolio:** Grid of cards featuring:
   - Project Title
   - Business Problem
   - Technical Solution
   - Key Result (ROI/Impact)
4. **Toolbox:** A curated list of technologies (SQL, Python, BigQuery, dbt, Power BI, Tableau).
5. **Contact/CTA:** "Let's collaborate" – LinkedIn link, GitHub link, and Email.
